# 箱入り娘の大家族

「箱入り娘の大家族」のWebゲームと、Go実装の高速ソルバーです。
通常の箱入り娘（5×4マス）の拡張版で、6×5マス・15駒のスライディングパズルです。

## 🎮 オンラインでプレイ

**[▶️ GitHub Pagesでプレイする](https://prime-oda.github.io/hakoiri_musume/)**

ブラウザで直接プレイできます。インストール不要です。

## ゲーム概要

### 基本ルール
- スライディングパズル
- 1つの駒を空きマスを伝ってスライドさせる。途中で何度向きを変えてもよく、連続した一続きの動きを1手と数える
- 目標：娘（2×2の赤い駒）を盤面下部の出口から脱出させる

### 盤面サイズ
- 6×5マス

## 駒の構成（合計15コマ）

| 駒名 | サイズ | 個数 | 説明 |
|------|--------|------|------|
| 娘 | 2×2 | 1 | 脱出させる目標の駒 |

| 母 | 1×2 | 1 | 縦長の駒 |
| 大番頭 | 4×1 | 1 | 横長の大きな駒 |
| 手代 | 1×1 | 1 | 小さな駒 |
| 女中 | 2×1 | 1 | 横長の駒 |
| 番頭 | 2×1 | 1 | 横長の駒 |
| 祖父 | 2×1 | 1 | 横長の駒 |
| 祖母 | 2×1 | 1 | 横長の駒 |
| 番犬 | 1×1 | 1 | 小さな駒 |
| 兄嫁 | 1×1 | 1 | 小さな駒 |
| 丁稚 | 1×1 | 3 | 小さな駒 |

## Webゲーム（web/）

### ローカル実行
```bash
cd web
python3 -m http.server 8000
# http://localhost:8000 をブラウザで開く
```

### 操作
- **クリック / タップ**: 移動先が一つだけの駒は自動移動
- **ドラッグ**: 複数の移動先がある場合はドラッグで移動先を選択
- モバイル（タッチ）操作にも対応

## Goソルバー（cmd/solver）

### ビルドと実行
```bash
# ビルド
go build -o solver ./cmd/solver

# 実行（デフォルトはBFS）
./solver

# アルゴリズムを指定
./solver -algo bidirectional

# 古典の箱入り娘（4×5・10駒）を解く
go build -tags classic -o solver_classic ./cmd/solver
./solver_classic
```

### 主なフラグ
| フラグ | 説明 | デフォルト |
|--------|------|------------|
| `-algo` | `bfs` / `bidirectional` | `bfs` |
| `-max-states` | 探索する最大状態数 | 10億 |
| `-max-time` | 探索の最大時間 | 30分 |
| `-verbose` | 進捗を表示 | false |
| `-bench` | ベンチマークモード | false |

実行が成功すると `solve.txt` に手順が書き出されます。
